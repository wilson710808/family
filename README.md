# 🏠 家族中心网站

一个专为全年龄段设计的家族社交平台，从小学三年级学生到七十多岁老人都能轻松使用。

## ✨ 核心功能

### 🎯 面向全年龄段的易用性设计
- **超大字体**：所有文字都使用更大字号，老人小孩都能轻松看清
- **大字模式**：一键切换，全站字體放大 1.5 倍
- **超大按钮**：最小点击区域 60px，避免误触，适合手指不灵活的用户
- **超大输入框**：20px 字号输入，打字更轻松
- **高对比度**：色彩对比度符合 WCAG 无障碍标准，视力不好也能看清
- **头像识别**：注册时选择头像，一看头像就知道是谁，方便家人识别
- **简单清晰的界面**：没有复杂的操作，每一步都有明确提示
- **图标+文字双标识**：所有功能都有图标和文字双重提示，不认字也能看懂
- **首次引导**：6 步驟新手指引，首次使用時自動展示

### 👥 用户系统

## 🚀 可插拔插件架构

本项目采用可插拔插件设计，新增功能都以插件形式加入，可以通过环境变量独立控制启用/禁用：

| 插件 | 默认 | 禁用环境变量 |
|------|------|--------------|
| 生日提醒 | 启用 | `PLUGIN_BIRTHDAY_REMINDER=false` |
| 家族相册 | 启用 | `PLUGIN_FAMILY_ALBUM=false` |
| 家族日历 | 启用 | `PLUGIN_CALENDAR=false` |
| 成长专栏 | 启用 | `PLUGIN_GROWTH_COLUMN=false` |
| 家族管家 | 启用 | `PLUGIN_FAMILY_BUTLER=false` |

插件特点：
- 独立目录结构，不影响核心代码
- 独立数据库表（所有表使用 `plugin_` 前缀）
- 禁用不需要修改代码，只改环境变量
- 完全删除可直接删除插件目录和对应数据表

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖
```bash
cd family-website
npm install
```

### 开发模式运行
```bash
npm run dev
```
访问 http://localhost:3000 即可查看网站

### 生产环境构建
```bash
npm run build
npm start
```

## 🎨 设计理念
### 针对老年人的优化
- 没有复杂的手势操作，都是简单的点击
- 没有闪烁、动画等可能引起眩晕的效果
- 所有可点击元素都有明确的边框和反馈
- 错误提示用大字体和表情符号，清晰易懂
- 操作步骤少，尽量一步到位
- 大字模式一键切换，更大字體顯示
- 语音输入，麦克风说话就能输入文字

### 针对儿童的优化
- 界面色彩明亮友好
- 图标生动有趣
- 操作简单，容易理解
- 没有复杂的表单
- 可爱的表情贴图互动

## 📝 使用流程
1. **注册账号**：选择头像 → 输入姓名 → 输入邮箱 → 设置密码 → 如果有推荐码请输入 → 完成注册
2. **创建家族**：登录后点击"创建家族"，输入家族名称，系统自动生成推荐码
3. **邀请成员**：把推荐码发给家人，家人注册时输入推荐码就能申请加入
4. **审核加入**：管理员在家族页面审核申请，同意后即可加入
5. **发布公告**：管理员可以在公告栏发布重要通知
6. **留言互动**：在留言板分享生活点滴
7. **实时聊天**：和家族成员在线聊天，支持发送表情贴图
8. **添加兴趣**：在个人设置中添加兴趣标签，方便连结同好
9. **切换主题**：点击导航栏太阳/月亮图标切换深色/浅色模式
10. **大字模式**：点击导航栏 A+ 图标开启大字模式
